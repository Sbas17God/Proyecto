# Proyecto
Proyecto de base de datos

# Descarga proyecto
Git clone "https://github.com/AlgorithmQuantum/Proyecto"

# Instala dependencias

- ## Crear el entorno virtual
    python -m venv nombre_del_entorno

- ## Activar el entorno
    nombre_del_entorno\Scripts\activate

- ## Para desactivar
    deactivate

- ## instalación del requerimientos
    pip install -r requirements.txt

# Ejecucion del programa
Una vez instaladas las depencencias y segurando de activar el entorno virtual, ejecutar los siguientes comandos:

    cd ./backend
    python app.py
